← Ver términos A-M
N
Node.js

Entorno de ejecución para JavaScript del lado del servidor que permite construir aplicaciones web escalables y en tiempo real.

O
Open Source

Software cuyo código fuente está disponible públicamente para ser estudiado, modificado y distribuido libremente.

P
Python

Lenguaje de programación de alto nivel conocido por su simplicidad y legibilidad, ampliamente usado en ciencia de datos e IA.

Q
QA (Quality Assurance)

Proceso para asegurar que un producto de software cumple con los estándares de calidad establecidos.

R
React

Biblioteca de JavaScript para construir interfaces de usuario interactivas y componentes web reutilizables.

S
SQL

Lenguaje estándar para gestionar y manipular bases de datos relacionales, fundamental en el desarrollo de aplicaciones.

T
TypeScript

Superconjunto de JavaScript que añade tipado estático opcional y otras características avanzadas para desarrollo.

U
UX (User Experience)

Disciplina que se centra en la experiencia total del usuario al interactuar con un producto digital.

V
VirtualBox

Software de virtualización para ejecutar múltiples sistemas operativos en una misma máquina física.

W
WebAssembly

Formato de código binario que permite ejecutar aplicaciones de alto rendimiento en navegadores web.

X
XML

Lenguaje de marcado extensible utilizado para almacenar y transportar datos de forma estructurada.

Y
YAML

Formato de serialización de datos legible por humanos, usado frecuentemente en archivos de configuración.

Z
Zero-Day

Vulnerabilidad de seguridad recién descubierta que los desarrolladores no han tenido tiempo de parchear.
